🗂️ HDFS Data Locations
# Input
/bda/raw/reviews/ ← raw Amazon review data

# Processed
/bda/processed/als_ratings/ ← userId, itemId, rating
/bda/processed/item_index/ ← asin ↔ itemId mapping
/bda/processed/bert_items/ ← asin, combinedText, avgRating

# Models
/bda/models/als_model/ ← ALS collaborative filter
/bda/models/word2vec_model/ ← W2V text embeddings
/bda/models/word2vec_embeddings/ ← item embedding vectors
/bda/models/gbt_ranker_model/ ← GBT hybrid ranker
/bda/models/hybrid_recommendations/ ← final top-10 per user
🔄 Retraining Guide
When new data arrives → run all steps in order.
When only refreshing recommendations → run Step 5 only.
# Build JAR (Windows CMD)
sbt clean assembly
docker cp target\scala-2.12\bda-project-assembly-1.0.jar spark-master:/bda-project-assembly-1.0.jar

# Submit job (inside spark-master)
/opt/spark/bin/spark-submit --class GenerateRecommendations \
  --master spark://spark-master:7077 \
  --conf spark.executor.memoryOverhead=1g \
  /bda-project-assembly-1.0.jar
